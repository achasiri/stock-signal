# STOCK·SIGNAL 📈
AI-powered daily stock analysis — Top 3 buys & avoids for low-moderate risk investors.

---

## 🚀 Deploy to Vercel in 5 Steps

### Step 1 — Deploy on Vercel
1. Go to **https://vercel.com**
2. Click **"Sign Up"** → choose **"Continue with GitHub"**
3. Click **"New Project"**
4. Find and import your `stock-signal` repo
5. Vercel auto-detects Vite — just click **"Deploy"**
6. Wait ~60 seconds ✅

### Step 2 — Get Your Live URL
Vercel gives you a URL like:
```
https://stock-signal-achasiri.vercel.app
```
That's it — your app is live!

---

## 🔧 How It Works
- On load, Claude searches live web news for trending stocks
- Analyzes market sentiment, earnings signals, and sector momentum
- Returns top 3 **BUY** picks and top 3 **AVOID** picks
- Hit **Refresh** anytime for a fresh analysis
