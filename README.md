# STOCK·SIGNAL 📈
AI-powered daily stock analysis — Top 3 buys & avoids for low-moderate risk investors.

---

## 🚀 Deploy to Vercel in 5 Steps

### Step 1 — Create a GitHub Account
1. Go to **https://github.com/signup**
2. Sign up with username `achasiri` and your email

### Step 2 — Create a New Repository
1. Click the **+** icon (top right) → **New repository**
2. Name it: `stock-signal`
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload These Files
1. In your new repo, click **"Add file"** → **"Upload files"**
2. Drag and drop the entire `stock-signal` folder contents:
   ```
   stock-signal/
   ├── index.html
   ├── package.json
   ├── vite.config.js
   ├── vercel.json
   ├── .gitignore
   └── src/
       ├── main.jsx
       ├── App.jsx
       └── index.css
   ```
3. Click **"Commit changes"**

### Step 4 — Deploy on Vercel
1. Go to **https://vercel.com**
2. Click **"Sign Up"** → choose **"Continue with GitHub"**
3. Click **"New Project"**
4. Find and import your `stock-signal` repo
5. Vercel auto-detects Vite — just click **"Deploy"**
6. Wait ~60 seconds ✅

### Step 5 — Get Your Live URL
Vercel gives you a URL like:
```
https://stock-signal-achasiri.vercel.app
```
That's it — your app is live!

## 🔧 How It Works
- On load, Claude searches live web news for trending stocks
- Analyzes market sentiment, earnings signals, and sector momentum
- Returns top 3 **BUY** picks and top 3 **AVOID** picks
- Hit **Refresh** anytime for a fresh analysis
